# 🕹️ ZK-Dash

A mini endless runner game themed around zk-proofs and Succinct's SP1 prover.

🎮 **Play now:**  
👉 [https://dlkabks.github.io/ZK-Dash/](https://dlkabks.github.io/ZK-Dash/)

## 💡 Game Details

- Avoid invalid operations  
- Complete recursive proof quests  
- Built with p5.js
